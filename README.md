# PRODIGY_WD_05 — Weather App

## Task
Build a web page that fetches weather data from a weather API based on the user's location or a user-inputted location. Display current weather conditions, temperature, and other relevant info.

## Tech Used
- HTML5
- CSS3
- Vanilla JavaScript (Fetch API, Geolocation API)
- **Open-Meteo API** — free, no API key required (geocoding + forecast endpoints)

## Features
- Search weather by typing any city name
- "Use my location" button (📍) using the browser Geolocation API
- Displays temperature, feels-like, humidity, wind speed, precipitation
- Weather icon + description mapped from WMO weather codes
- Error handling for invalid city names / denied location access

## How to Run
1. Clone this repo
2. Open `index.html` in any browser
3. No API key setup needed — Open-Meteo is free and keyless
